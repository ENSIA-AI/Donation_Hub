#  DONIFY.dz

##  Project Goals
The goal of **DONIFY.dz** is to create a simple digital bridge where real organizations can post announcements seeking help, and users can see them and contribute through donations. 
It aims to encourage community support and make the process of helping others faster and more transparent.

---

##  Overview
**DONIFY.dz** is a web platform designed to connect **donors** with **organizations** in need.  
It allows verified organizations to create announcements for help, while users can browse them and make donations easily through an online form.  

The project is part of an academic web development initiative.

---

##  Technologies Will be Used
- **HTML** – for page structure  
- **CSS** – for layout and design  
- **JavaScript** – for client-side interactivity  
- **PHP** – for backend logic and data handling  
- **React** – for creating dynamic user interfaces
